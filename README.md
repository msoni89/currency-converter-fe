
## Currency Converter
### Front End:Implement React component - currency calculator.
```
The component should implement the possibility of converting different currencies using
current exchange rate. Also, this component should draw swing chart (or candle chart) for
current pair of currencies for a selected time frame (e.g. intraday chart - with 5 minutes interval,
daily/weekly chart - with 60 minutes interval, monthly chart - with 1-day interval) 
```
### Example: google converter picture

The component should fetch data from the back-end described below.
### Supported currencies: EUR, USD, GBP

### Build
```
- npm run build
- npm start
```

### Dependency - Backend spring boot server (https://github.com/msoni89/currency-converter.git)
